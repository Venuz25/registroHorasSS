# registroHorasSS

Aplicación web diseñada para facilitar el registro, seguimiento y administración de las horas de servicio social de los estudiantes. El sistema permite a los alumnos registrar sus actividades diarias mediante una interfaz de calendario, mientras que los administradores cuentan con un panel de control para supervisar el progreso y gestionar el estado de los usuarios.
